\section{概要}
公理集合 $\mathcal{A}$に対して、構文的変換$\mathcal{T}$を定義し、$\mathcal{T}{\mathcal{A}}$が公理集合となる。

この構文的変換を公理の変換と呼ぶ。

$\mathcal{T}$は論理式ではなく、構文的な変換なので、$\mathcal{A}$と$\mathcal{T}{\mathcal{A}}$に$\mathcal{T}$を加えても証明はできない。そもそも、$\mathcal{T}$は論理式ではない。
